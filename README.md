# computils.js

Low-level programming tools in plain JavaScript. Browser-first. This does not replace the official NVIDIA CUDA Toolkit, Vulkan Toolkit, AMD HIP Toolkit, Intel Graphics Compiler, Intel oneAPI Toolkit, Microsoft Visual C++, Apple Xcode, LLVM toolchain and other official tools in certain situations.

## Requirements

- A modern browser with ES6/ES7 support, or specifically:
  - Chrome 72.0.x.xxxx
